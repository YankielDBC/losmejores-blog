# SPEC.md - LosMejores.blog Landing Page

## Project Overview
- **Name:** LosMejores.blog - Landing Page Principal
- **Type:** Static marketing landing page
- **Core Functionality:** Presentación de quienes somos como reviewers, metodología, y qué encontrarán en el sitio
- **Target Users:** Personas buscando recomendaciones de productos

## Design Philosophy
Following taste-frontend principles:
- DESIGN_VARIANCE: 8 (Asymmetric, artsy)
- MOTION_INTENSITY: 6 (Cinematic animations)
- VISUAL_DENSITY: 4 (Art Gallery - espacioso y premium)
- NO emojis, NO AI purple, NO centered layouts

## UI/UX Specification

### Layout Structure
- **Header:** Sticky nav con logo + links internos
- **Hero:** Split screen - texto izq, imagen derecha asimétrica
- **About:** Grid asimétrico con nuestra historia
- **Metodología:** Cards con icons (no emojis)
- **Qué Encontrarán:** Bento grid asimétrico
- **CTA:** Newsletter signup
- **Footer:** Links, social, disclosure

### Responsive Breakpoints
- Mobile: < 768px (single column)
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Visual Design
- **Colors:**
  - Primary: Deep charcoal (#1a1a2e)
  - Accent: Electric teal (#00d4aa)
  - Background: Off-white (#fafafa)
  - Text: Dark gray (#2d2d2d)
- **Typography:**
  - Headlines: Geist or Satoshi (bold, tight tracking)
  - Body: Geist (regular)
- **Spacing:** Generous, art gallery feel

### Components
1. Navbar con blur effect
2. Hero con staggered reveal animation
3. About cards con hover lift
4. Methodology steps con icons (Lucide)
5. Stats counter animation
6. Newsletter form
7. Footer con links
